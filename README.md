## Langages utilisés
    - HTML

## Lancer l'application
    - Lancer un serveur pour le php, type Apache
    - Lancer un serveur Msdql
    - En local j'utilse MAMP
        - serveur : "localhost"
        - username : "root"
        - password : "root"
    - La base de donnée est fournie dans le repertoire database
